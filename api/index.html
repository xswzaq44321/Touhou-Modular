Boss 的 function

在遊戲內console，你可以輸入"print(boss.help)"來顯示所有boss可用函式。
也可以輸入"print(boss.explain.<function-name>)"來看詳細的解說。

boss.evenN_way(string load_type, string bullet_type, int even_N, float interval, float speed, float degree = -90, float initial_radius = 0, float direction = 0)
shoot even number danmaku
  • load_type: type of visual effect, usually ends with _load
  • bullet_type: type of bullet
  • interval: the angle(deg) between every bullet
  • speed: speed of bullet
  • degree: angle(deg) of the middle bullet
  • initial_radius: the danmaku circle's radius
boss.oddN_way(string load_type, string bullet_type, int odd_N, float interval, float speed, float initial_radius = 0)
shoot odd number danmaku, the middle bullet will automatically track player
  • load_type: type of visual effect, usually ends with _load
  • bullet_type: type of bullet
  • odd_N: how many bullets per circle (odd num)
  • interval: the angle(deg) between every bullet
  • speed: speed of bullet
  • initial_radius: the danmaku circle's radius
boss.radiate(string load_type, string bullet_type, int N, float speed, float degree = -90, float initial_radius = 0)
radiate danmaku
  • load_type: type of visual effect, usually ends with _load
  • bullet_type: type of bullet
  • N: how many bullets per circle
  • speed: speed of bullet
  • degree: angle(deg) of the first bullet
  • initial_radius: the danmaku circle's radius
boss.shoot(string load_type, string bullet_type, float posx, float posy, float speed, float degree)
for single bullet
  • load_type: type of visual effect, usually ends with _load
  • bullet_type: type of bullet
  • posx, posy: initial position of bullet
  • speed: speed of bullet
  • degree: the direction bullet flys (deg)
boss.laser(string laser_type, float str_pointx, float str_pointy, float str_pointz, float str_angle, float time, float angle = 0, float angular_speed = 0)
for laser that sticks with the transform
  • laser_type: type of laser
  • str_pointx, str_pointy, str_pointz: where the laser starts
  • str_angle: initial angle(deg) of the laser
  • time: how long will the laser last(doesn't include the time it generates or rotates
  • angle: how much will you want to rotate (deg)
  • angular_speed: rotate at speed (deg/sec), counterclockwise as positive
